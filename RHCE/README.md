# Pluralsight Red Hat Certified Engineer Path
This 16 hour, self-paced programme covers how to automate Linux administration with the Red Hat Ansible Automation Platform. In addition, these courses cover the objectives and competencies of the Red Hat Certified Engineer (RHCE) exam and can help you prepare.
## Module Notes
- [What's New In RHEL 9](What's%20New%20In%20RHEL%209.md)
- Linux Administration with Ansible: Getting Started with Ansible Automation
- Linux Administration with Ansible: Advanced Ansible Automation
- Linux Administration with Ansible: Writing Ansible Playbooks
